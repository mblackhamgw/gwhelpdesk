Welcome to the GroupWise Helpdesk Application - {{ request.session.adminname}}

Use the left menus to search for GroupWise Users, or performing site administration functions.

Edit the GroupWise Configuration settings to allow the application to connect to your GroupWise Admin Service. The Administrator must be a GroupWise System Administrator.

There are currently 4 administrator roles:

  • Administrator - This role allows all functions of the application and is also used to add other administrators.
  • Helpdesk - This role allows modification, adds, deletes and password change for all GroupWise user objects.
  • Helpdesk Lite - This role allows modification and password change, but not add or delete rights for all GroupWise user objects.
  • Password - This role is restriction to only GroupWise user password changes.

If a GroupWise user account is associated to a eDir or Active Directory account, most of the details for that user will be read-only. If the associated user's Post Office is set for LDAP Authentication, the change password option will be disabled.
